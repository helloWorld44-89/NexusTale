# NexusTale Brand Assets  
  
Official branding package for **NexusTale** — Open source AI-assisted novel writing app (sci-fi & fantasy + git versioning).  
  
## 🎨 Color Palette  
  
**Primary Colors**  
- Electric Cyan (Tech / AI / Nexus): `#00F0FF`  
- Mystic Purple (Fantasy / Magic): `#9F4BFF`  
- Golden Accent (Storytelling / Energy): `#F4C95D`  
  
**Backgrounds**  
- Dark Mode: `#0F0F1A` or `#12121A`  
- Light Mode: `#FAF9F6` or `#FFFFFF`  
  
**Text**  
- Dark mode text: `#E0E0FF`  
- Light mode text: `#1A1A2E`  
  
## 🔤 Typography  
  
**Logo Font (exact match recommended)**  
- Modern sans-serif with slight futuristic weight  
- Suggested fonts (free): **Inter**, **SF Pro**, **Poppins**, or **Orbitron** (for "Nexus")  
  
**Usage**  
- Headings: Inter Bold / Poppins Semi-Bold  
- Body: Inter Regular  
  
## 📁 File Usage Guide  
  
| File | Best Use |  
|------|----------|  
| Primary-Logo-Dark.png | Website header, GitHub repo, dark UI |  
| Primary-Logo-Light.png | App Store screenshots, light docs |  
| Icon-Mark-Dark.png | Favicon (32×32, 180×180), social avatars |  
| Icon-Mark-Light.png | Light-themed favicons |  
| App-Icon-Dark.png | iOS, Android, Mac App Store |  
| App-Icon-Light.png | Alternative store listings |

```html  
<link rel="icon" href="icon-mark-dark-32.png" media="(prefers-color-scheme: dark)">
